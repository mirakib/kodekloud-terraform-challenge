# Create EBS Volume Using Terraform

For this task, create an AWS EBS volume using Terraform with the following requirements:
- Name of the volume should be `xfusion-volume`.
- Volume type must be `gp3`.
- Volume size must be `2 GiB`.
- Ensure the volume is created in `us-east-1`.

**`main.tf`**

```HCL
resource "aws_ebs_volume" "xfusion_volume" {
  availability_zone = "us-east-1a"
  size              = 2
  type              = "gp3"

  tags = {
    Name = "xfusion-volume"
  }
}
```

## How to apply

```bash
terraform init
terraform plan
terraform apply
```
